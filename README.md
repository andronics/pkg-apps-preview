# File Preview - Quick file content viewer

**Package**: `@apps/preview`
**Version**: 0.1.0
**Repository**: `pkg-apps-preview`

## Overview

File Preview - Quick file content viewer

## Installation

### Prerequisites

No external dependencies required.

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @apps/preview

# Or install from local source
cd ~/.pkgs
stow preview
```

## Usage

### Application

File Preview - Quick file content viewer

Run the application to see usage instructions.

## Configuration

Configuration files are typically stored in:
- `~/.config/preview/` - User configuration
- `~/.local/share/preview/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @apps/preview

# Or manual unstow
cd ~/.pkgs
stow -D preview
```
